## 一、交互动画改进

### 1.1 歌曲行 hover 效果
```css
.track-row{position:relative;transition:all .2s var(--smooth)}
.track-row:hover{background:var(--hover);transform:translateX(4px)}
.track-row::before{content:"";position:absolute;left:0;top:50%;transform:translateY(-50%);width:0;height:60%;background:var(--accent);border-radius:0 3px 3px 0;transition:width .2s var(--spring)}
.track-row:hover::before{width:3px}
```

### 1.2 来源彩色标签
- 酷狗：粉色 `#FF6B9D`
- 网易云：红色 `#FF4757`
- B站：青色 `#00D2D3`

### 1.3 侧边栏品牌呼吸动画
```css
.sb-brand svg{animation:breathe 3s ease-in-out infinite}
@keyframes breathe{0%,100%{transform:scale(1);filter:drop-shadow(0 0 4px rgba(var(--accent-rgb),.2))}50%{transform:scale(1.05);filter:drop-shadow(0 0 8px rgba(var(--accent-rgb),.4))}}
```

### 1.4 进度条时间预览
- hover 进度条时，手柄上方显示当前预览时间

### 1.5 歌词面板过渡动画
```css
.lyric-panel{transition:width .4s var(--spring),opacity .3s}
.lyric-panel.hidden{width:0;opacity:0;overflow:hidden}
```

### 1.6 空状态微浮动
```css
.empty-state svg{animation:float 3s ease-in-out infinite}
@keyframes float{0%,100%{transform:translateY(0)}50%{transform:translateY(-8px)}}
```

### 1.7 播放按钮切歌弹跳
```css
.play-btn.playing{animation:pulse-glow 2.5s ease-in-out infinite}
@keyframes pulse-glow{0%,100%{box-shadow:0 2px 12px rgba(var(--accent-rgb),.3)}50%{box-shadow:0 2px 24px rgba(var(--accent-rgb),.55)}}
```

### 1.8 搜索框图标
- 在搜索输入框左侧添加搜索图标

### 1.9 Tab 切换动画
- JS: switchTab 时添加 fadeSlideIn 动画到内容区

## 四、自定义标题栏

### 当前问题
使用 Windows 默认标题栏，与软件风格不匹配。

### 方案
1. **main.go** — 使用 WebView2 的无边框窗口模式
2. **HTML** — 添加自定义标题栏（带最小化/最大化/关闭按钮）
3. **CSS** — 标题栏匹配蓝色玻璃态风格

### 标题栏设计
```
┌────────────────────────────────────────────────────────┐
│ 🎵 MelodyV3          [─] [□] [×]                      │ ← 自定义标题栏
├────────────────────────────────────────────────────────┤
│ 侧边栏 | 内容区                                        │
│ ...                                                    │
├────────────────────────────────────────────────────────┤
│ 播放栏                                                 │
└────────────────────────────────────────────────────────┘
```

### CSS
```css
.titlebar{position:fixed;top:0;left:0;right:0;height:36px;display:flex;align-items:center;justify-content:space-between;padding:0 12px;background:var(--card);border-bottom:1px solid var(--border);z-index:100;-webkit-app-region:drag}
.titlebar-btns{display:flex;gap:4px;-webkit-app-region:no-drag}
.titlebar-btn{width:30px;height:24px;border:none;border-radius:4px;background:transparent;color:var(--text2);cursor:pointer;display:flex;align-items:center;justify-content:center;transition:background .15s}
.titlebar-btn:hover{background:var(--hover)}
.titlebar-btn.close:hover{background:#e81123;color:#fff}
```

### main.go 改动
- 使用 `webview2.New()` 的无边框模式
- 通过 WebView2 的 IPC 与 Go 后端通信实现最小化/最大化/关闭
