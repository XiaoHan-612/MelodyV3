# MelodyV3 全面改造方案 v3

## 设计理念

**参考 Mineradio + 苹果动画哲学**，保持蓝色系，用纯 Canvas2D（不用 Three.js）实现粒子效果，重点打磨动画细节。

### 核心原则
1. **动画决定体验** — 苹果的流畅感来自 spring 曲线 + momentum + 元素联动
2. **性能优先** — 用 Canvas2D 替代 Three.js，120fps 不卡顿
3. **细节为王** — 每个按钮 hover、每个页面切换、每次状态变化都有动画
4. **蓝色系** — 主色 `#007AFF`（iOS 蓝），辅色 `#409CFF`，深色背景 `#0A0E1A`

### 1.4 进度条拖拽优化

```javascript
// 拖拽时：音频暂停 seek，松手后恢复
// 视觉反馈：进度条变为 accent 色，手柄放大
var isDragging = false;
var wasPlaying = false;

function onProgressMouseDown(e) {
  isDragging = true;
  wasPlaying = !audio.paused;
  audio.pause();  // 拖拽时暂停，避免杂音
  seekTo(e);
  $("progressBar").classList.add("dragging");
}

function onProgressMouseMove(e) {
  if (!isDragging) return;
  seekTo(e);
  // 实时更新时间显示
  var p = getSeekPercent(e);
  $("timeDisplay").textContent = formatTime(p * audio.duration) + " / " + formatTime(audio.duration);
}

function onProgressMouseUp() {
  if (!isDragging) return;
  isDragging = false;
  if (wasPlaying) audio.play();
  $("progressBar").classList.remove("dragging");
}
```

## 四、全屏歌词（粒子 + 渐变文字 + 逐字高亮）

### 4.1 背景
- 深蓝渐变 `#0A0E1A → #0E1425`
- 800+ 环境粒子做星空（Canvas2D，不用 Three.js）
- 封面模糊层（CSS filter: blur + saturate）

### 4.2 歌词文字
```css

  font-weight: 700;
  /* 渐变色文字 */

  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
  /* 三层发光 */
  filter: drop-shadow(0 0 12px rgba(0,122,255,.4))
          drop-shadow(0 0 30px rgba(0,122,255,.2))
          drop-shadow(0 0 60px rgba(0,122,255,.1));
  /* 3D 入场动画 */
```

### 4.3 逐字高亮（Canvas2D）
```javascript
function renderKaraoke(ctx, text, cx, cy, size, progress) {
  ctx.font = `700 ${size}px "Noto Sans SC", sans-serif`;
  ctx.textAlign = "center";
  ctx.textBaseline = "middle";

  var chars = text.split("");
  var metrics = ctx.measureText(text);
  var charW = metrics.width / chars.length;
  var startX = cx - metrics.width / 2;
  var litIdx = Math.floor(progress * chars.length);

  for (var i = 0; i < chars.length; i++) {
    var x = startX + i * charW + charW / 2;
    var lit = i <= litIdx;

    ctx.save();
    if (lit) {
      // 已播放：亮白 + 青色发光
      ctx.fillStyle = "#E8F8FF";
      ctx.shadowColor = "rgba(0,122,255,.7)";
      ctx.shadowBlur = 16;
      ctx.font = `700 ${size * 1.02}px "Noto Sans SC", sans-serif`;
    } else {
      // 未播放：暗蓝灰
      ctx.fillStyle = "rgba(255,255,255,.18)";
      ctx.shadowBlur = 0;
    }
    ctx.fillText(chars[i], x, cy);
    ctx.restore();
  }
}
```

## 五、主界面音轨

### 全宽竖条可视化器（播放栏正上方）
```javascript
function drawWaveformBars(ctx, w, h, freqData) {
  var barCount = 100;
  var barW = w / barCount;
  for (var i = 0; i < barCount; i++) {
    var fi = Math.floor((i / barCount) * freqData.length * 0.7);
    var fv = freqData[fi] / 255;
    var barH = Math.max(2, fv * h * 0.85);
    var x = i * barW + 1;
    var y = h - barH;

    // 渐变竖条
    var grad = ctx.createLinearGradient(x, h, x, y);
    grad.addColorStop(0, "rgba(0,122,255,.08)");
    grad.addColorStop(0.5, "rgba(0,122,255,.5)");
    grad.addColorStop(1, "rgba(64,156,255,.8)");
    ctx.fillStyle = grad;
    ctx.fillRect(x, y, barW - 2, barH);

    // 顶部光点
    if (fv > 0.12) {
      ctx.beginPath();
      ctx.arc(x + (barW - 2) / 2, y, 1.5, 0, 6.28);
      ctx.fillStyle = `rgba(128,200,255,${fv * .7})`;
      ctx.fill();
    }
  }
}
```

## 六、歌词进入/退出完整动画流程

### 进入（3步动画）
1. **0-300ms**：背景渐显 + 环境粒子开始浮动
2. **300-700ms**：歌词文字从 `perspective(800px) translateZ(-40px) rotateX(5deg)` spring 回到正常位置 + `blur(8px)` → 清晰
3. **700-1200ms**：火花粒子从文字中心向外扩散 + 火花闪烁

### 退出（2步动画）
1. **0-400ms**：火花粒子收缩回文字中心 + 文字轻微缩小
2. **400-700ms**：文字向远处飞走 `translateZ(30px) rotateX(-3deg)` + `blur(6px)` → 消失

### 歌词行切换
- 旧行：`stage-out` 动画（400ms）→ 向上飞走 + 模糊
- 新行：`stage-in` 动画（900ms）→ 从远处 spring 回来 + 3D 旋转

## 七、性能优化

### Canvas2D 替代 Three.js
- 环境粒子：800 个简单圆点，每帧只更新位置 + 透明度
- 火花粒子：100-150 个，life 衰减后回收
- 歌词文字：直接 Canvas2D fillText，不用纹理贴图
- 音轨竖条：100 根简单矩形

### 帧率控制
```javascript
var lastFrame = 0;
var targetFPS = 60;
var frameInterval = 1000 / targetFPS;

function animationLoop(timestamp) {
  if (timestamp - lastFrame >= frameInterval) {
    lastFrame = timestamp;
    draw();  // 只在目标帧率时绘制
  }
  requestAnimationFrame(animationLoop);
}
```

### 按需绘制
- 只在 `state.lyricsOpen` 时绘制全屏歌词 canvas
- 只在 `state.isPlaying` 时更新音轨
- 背景粒子可以降低更新频率（每2帧更新一次位置）
