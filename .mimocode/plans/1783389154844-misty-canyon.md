# MelodyV3 UI 全面重设计 v2

## 设计目标
参考 YesPlayMusic / AlgerMusicPlayer / SPlayer 的设计理念，对 MelodyV3 进行**整体布局 + 视觉风格 + 歌词体验**的全面重构。

**主色调：蓝色系** — 浅色 `#007AFF`，暗色 `#0A84FF`

---

## 六、沉浸式歌词（核心亮点）

### 设计理念
**取消固定歌词面板**，改为：点击播放栏封面 → 进入**全屏沉浸式歌词页面**，类似 AlgerMusicPlayer 的沉浸式体验。

### 触发方式
- 点击播放栏封面图 → 打开沉浸式歌词
- 快捷键 `L` / `F` → 打开沉浸式歌词
- 点击左上角 `←` 或按 `Esc` → 返回

### 全屏歌词页面布局

```

│                                                          │
```

**背景效果**：
- 封面图片放大 + 高斯模糊 `filter: blur(80px) brightness(.15) saturate(1.3)`
- 暗色渐变叠加 `linear-gradient(180deg, rgba(0,0,0,.4) 0%, rgba(0,0,0,.7) 50%, rgba(0,0,0,.85) 100%)`
- 整体色调跟随封面主色（已有 `extractCoverColor`）

### 歌词逐字渐变扫描（核心炫酷效果）

当前已有 `--progress` CSS 变量实现渐变扫描，需要**增强**：

```css
.lyric-line.active .text {
  font-size: 24px;
  font-weight: 700;
  color: transparent;
  background: linear-gradient(90deg,
    var(--accent) 0%,
    var(--accent) var(--progress, 0%),

    rgba(255,255,255,.35) 100%

  -webkit-background-clip: text;
  background-clip: text;
  filter: drop-shadow(0 0 20px var(--accent-bg));
  transition: font-size .4s cubic-bezier(.4,0,.2,1);
}
```

### 粒子/极光背景动效（保留并增强）

保留现有的 `drawGrandWave` 频谱动效，但做以下调整：
- 粒子颜色从红色改为蓝色：`rgba(10,132,255, alpha)`
- 极光条带改为蓝色渐变
- 中心光晕改为蓝色
- 整体降低强度，让歌词文字更突出

```javascript
// 修改 drawGrandWave 中的颜色
ctx.shadowColor = "rgba(10,132,255,0.6)";  // 原来是红色
// 渐变
grad.addColorStop(0, "rgba(10,132,255,0)");
grad.addColorStop(0.5, "rgba(64,156,255,0.7)");
grad.addColorStop(1, "rgba(10,132,255,0)");
```

### 歌词滚动行为
- 活跃行始终居中
- 滚动使用 `scrollIntoView({ behavior: "smooth", block: "center" })`
- 非活跃行渐隐 `opacity: .35`，活跃行全亮
- 上下各 2 行做渐变遮罩 `mask: linear-gradient(transparent 0%, #000 15%, #000 85%, transparent 100%)`

### 翻译歌词显示
- 翻译行紧跟在主歌词行下方
- 字号 13px，颜色 `rgba(255,255,255,.45)`
- 翻译开关按钮 `[译]` 在头部

## 九、动画清单

| 动画 | 效果 | 实现方式 |
|------|------|---------|
| 卡片入场 | 逐个淡入 + 上移 | CSS `animation: fadeSlideUp .3s ease both` + JS 动态 delay |
| 页面切换 | 内容区淡入 + 上移 | CSS `@keyframes fadeSlideIn` |
| 播放按钮呼吸 | 播放时蓝色光晕脉动 | CSS `@keyframes breathe` |
| 封面切换 | 缩小 → 淡出 → 新封面淡入 → 放大 | JS class toggle + CSS transition |
| 歌词高亮 | 活跃行放大 + 颜色渐变 | CSS transition `font-size .4s, color .3s` |
| 歌词渐变扫描 | 从左到右蓝色填充 | CSS `background: linear-gradient(var(--progress))` |
| 进度条手柄 | hover 时出现圆形拖拽点 | CSS opacity transition |
| 全屏歌词进入 | 从下方滑入 + 淡入 | CSS `@keyframes slideUpIn` |
| 侧边栏导航 | 点击时缩放反馈 | CSS `:active { transform: scale(.97) }` |
