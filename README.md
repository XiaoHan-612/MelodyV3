# MelodyV3 - 三源聚合音乐播放器

一个现代化的桌面音乐播放器，聚合酷狗、网易云、B站三大音乐平台。

## 功能特性

### 核心功能
- **三源聚合搜索** - 同时搜索酷狗、网易云、B站，结果按来源分组排序
- **智能播放** - 自动获取最佳播放链接，失败时自动切换其他源（Fallback）
- **歌词同步** - 逐字渐变高亮显示，自动滚动，支持歌词偏移调节
- **歌词翻译** - 自动显示网易云翻译歌词（如有），可一键开关
- **歌单管理** - 本地歌单持久化存储，支持添加/移除/去重

### 界面设计
- **现代化 UI** - 毛玻璃效果，流畅动画过渡
- **深色/浅色主题** - 一键切换，偏好自动保存
- **封面主题色自适应** - 播放歌曲时从封面提取主色调，界面颜色跟随变化
- **全屏歌词** - 沉浸式歌词体验，带粒子/极光/径向波纹频谱动效

### 交互优化
- **快捷键支持**
  - `空格` - 播放/暂停
  - `←/→` - 快退/快进 5 秒
  - `↑/↓` - 调节音量
  - `M` - 静音切换
  - `L` - 歌词面板切换
  - `F` - 全屏歌词
  - `Esc` - 关闭全屏/菜单
- **鼠标操作**
  - 点击/拖拽进度条跳转（拖拽时音频不中断）
  - 滚轮调节音量
- **右键菜单** - 播放、下一首播放、添加到歌单、移除、复制歌曲名
- **搜索历史** - 自动记录最近 10 条搜索，点击快速重搜
- **播放队列** - 查看当前队列，支持拖拽排序和移除

### 播放模式
- 顺序播放 / 列表循环 / 单曲循环 / 随机播放

### 倍速播放
- 支持 0.5x、0.75x、1x、1.25x、1.5x、2x

### 音频增强
- 切歌渐入渐出（0.2s），过渡更自然
- Web Audio API 频谱可视化（底部波形 + 全屏粒子动效）

## 安装使用

### 直接运行
1. 下载 `MelodyV3.exe`
2. 双击运行即可（无需安装，依赖系统自带的 WebView2 运行时）

### 从源码编译
```bash
# 前置要求：Go 1.22+、Git、rsrc (go install github.com/akavel/rsrc@latest)
git clone <repository>
cd melodyv3
build.bat
```

## API 接口

### 搜索音乐
```
GET /api/search?keyword={关键词}&source={来源}
```
- `keyword`: 搜索关键词（必填）
- `source`: 音乐源（kg/ne/bl，可选，默认全部并发搜索）

### 获取播放链接（带自动切源）
```
GET /api/song/url?id={歌曲ID}&source={来源}
```

### 获取歌词
```
GET /api/song/lyric?id={歌曲ID}&source={来源}
```

### 按歌名搜索歌词（跨平台匹配）
```
GET /api/song/lyric/search?title={歌名}&artist={歌手}
```

### 歌单管理
```
GET  /api/playlist          # 获取歌单
POST /api/playlist          # 保存歌单（JSON body，上限 1MB）
```

## 常见问题

### Q: 为什么有些歌曲无法播放？
A: 可能是版权限制或链接失效，程序会自动尝试其他源。

### Q: 歌词不同步怎么办？
A: 点击歌词面板的 `-0.5s` / `+0.5s` 按钮手动调节偏移。

### Q: 如何备份歌单？
A: 歌单保存在 `~/.melody3_playlist.json`，可直接复制备份。

### Q: 封面颜色没变化？
A: 部分歌曲封面跨域受限或颜色太灰（饱和度低），会自动使用默认红色。
