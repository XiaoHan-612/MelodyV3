# MelodyV3 - 三源聚合音乐播放器

一个现代化的桌面音乐播放器，聚合酷狗、网易云、B站三大音乐平台。

## 功能特性

### 核心功能
- **三源聚合搜索** - 同时搜索酷狗、网易云、B站，结果去重排序
- **智能播放** - 自动获取最佳播放链接，失败时自动切换源
- **歌词同步** - 支持逐字歌词显示，自动滚动高亮
- **歌单管理** - 本地歌单存储，支持导入/导出

### 界面设计
- **现代化 UI** - 毛玻璃效果，流畅动画过渡
- **响应式布局** - 自适应窗口大小
- **深色/浅色主题** - 跟随系统设置（可扩展）

### 交互优化
- **快捷键支持**
  - `空格` - 播放/暂停
  - `←/→` - 快退/快进 5 秒
  - `↑/↓` - 调节音量
- **鼠标操作**
  - 点击进度条跳转
  - 拖拽进度条预览
  - 滚轮调节音量
- **右键菜单** - 添加到歌单、移除等操作

### 播放模式
- 顺序播放
- 列表循环
- 单曲循环
- 随机播放

### 倍速播放
- 支持 0.5x、0.75x、1x、1.25x、1.5x、2x

## 安装使用

### 直接运行
1. 下载 `MelodyV3.exe`
2. 双击运行即可（无需安装）

### 从源码编译
```bash
# 前置要求：Go 1.26+、Git
git clone <repository>
cd melodyv3
./build.bat
```

- **后端**: Go + net/http

## API 接口

### 搜索音乐
```
GET /api/search?keyword={关键词}&source={来源}
```
- `keyword`: 搜索关键词（必填）
- `source`: 音乐源（kg/ne/bl，可选，默认全部）

### 获取播放链接
```
GET /api/song/url?id={歌曲ID}&source={来源}
```

### 获取歌词
```
GET /api/song/lyric?id={歌曲ID}&source={来源}
```

### 歌单管理
```
GET  /api/playlist          # 获取歌单
POST /api/playlist          # 保存歌单
```

## 开发说明

### 添加新音乐源
1. 在 `api.go` 中实现 `MusicSource` 接口
2. 在 `searchAll` 函数中添加新源
3. 在前端添加对应的来源标签

### 自定义 UI
- 修改 `static/index.html` 中的 CSS 变量
- 调整颜色、圆角、阴影等样式

## 常见问题

### Q: 为什么有些歌曲无法播放？
A: 可能是版权限制或链接失效，程序会自动尝试其他源。

### Q: 歌词不同步怎么办？
A: 歌词解析依赖源数据质量，部分歌曲可能没有精确时间戳。

### Q: 如何备份歌单？
A: 歌单保存在 `~/.melody3_playlist.json`，可直接复制备份。
